# Cranium x EO Joint Venture - Investment Proposal

A professional, password-protected HTML presentation for the Board of Directors.

## Features

- ✅ Single, self-contained HTML file (works offline)
- ✅ Password protection (Username: Board, Password: #sta@1!9!2!1!)
- ✅ Interactive financial charts (NPV, Revenue, Expenses, Profit, Cashflow)
- ✅ Complete feature matrix (34 features)
- ✅ Ship count growth visualization
- ✅ Oldendorff branding and color theme
- ✅ Keyboard navigation (arrow keys)
- ✅ Full-screen presentation mode

## Deployment

### Option 1: Deploy to Vercel (Recommended)

1. Push this repository to GitHub
2. Import to Vercel (https://vercel.com/new)
3. Connect your creo.io domain
4. Done! Your site is live and always-on

### Option 2: Local Testing

```bash
npm run dev
# Visit http://localhost:3000
```
